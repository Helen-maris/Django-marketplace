# Сайт объявлений
Это учебный проект по созданию сайта объявлений. На данный момент реализована возможность добавлять информацию о продавце, добавлять и редактировать объявления, сортировать их по тэгам, просматривать только свои объявления. Есть возможность регистрации на сайте через Google. Доступ к некоторым страницам ограничен для неавторизированных пользователей. При помощи Celery реализована рассылка уведомлений подписчикам (например, на новые объявления определенной категории или объявления за определенный период)
